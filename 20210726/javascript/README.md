# 프렌즈 4 블록
https://programmers.co.kr/learn/courses/30/lessons/17679
## 구하는 것
지워지는 '블록'의 개수
## 주어진 것
- '판'의 높이, 폭 m, n
- '판'의 '배치' 정보 board
## 조건
### 블록이 지워진다
- 2 x 2 '영역'에 같은 '블록'이 '배치'되어 있으면 지워진다. 
- 한 블록이 여러 '영역'에 동시에 속하면, 모두 사라진다. 
### 떨어진다
- 중간에 빈공간이 생기지 않도록 떨어진다. 
### 연쇄
- 지워야할 블록이 없을때까지, 블록이 지워지고, 떨어지고를 반복한다.
