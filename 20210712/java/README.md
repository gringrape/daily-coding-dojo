# 리틀 프렌즈 사천성
https://programmers.co.kr/learn/courses/30/lessons/1836
## 구하는 것
- 규칙에 따라 타일을 모두 제거할 수 있는지 여부.
- 있다면, 어떤 순서로 제거하면 되는지.
## 주어진 것
- 게임판 크기 m, n
- 타일 정보 배열 board
## 조건
### 타일을 제거하는 규칙
- 경로 조건
- 경로는 한 번 이하로 꺾일 수 있다.
- 경로 중간에 장애물이 있어서는 안된다. 
### board
- . 빈칸
- * 막힌칸
    
## 계획
- 현재 상태에서 제거 가능한 모든 타일을 찾는다. 
- 각 타일에 대해서 다음 상태로 넘어간다.
