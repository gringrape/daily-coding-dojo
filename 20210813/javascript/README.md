# 카드 짝 맞추기
https://programmers.co.kr/learn/courses/30/lessons/72415
## 구하는 것
모든 `카드를 제거`하기 위한 `키 조작` 횟수의 최솟값
## 주어진 것
- 현재 `카드`의 `앞면` 배열 -> 4 x 4 격자
- `커서` 의 처음 위치
## 조건
### 카드의 제거
- 카드를 두장 선택했을때, `앞면`이 같으면 `제거`
### 커서
- 카드는 `커서`를 이용해 `선택` 가능.
- `커서` 는 유저가 선택한 위치.
- `커서` 는 방향키를 누른 방향으로 이동.
### 키 조작 (커서 이동, 선택)
- 상하좌우 방향키 이동 -> 한칸
- CTRL + 방향키 -> 빈칸을 건너뛰어 이동
- ENTER -> 카드를 선택
