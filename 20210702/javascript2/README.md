# 블록 이동하기
https://programmers.co.kr/learn/courses/30/lessons/60063

## 구하는 것
로봇이 (N, N) 위치까지 이동하는데 필요한 최소 시간

## 주어진 것
0 과 1 로 이루어진 board

## 조건
### 0 과 1 의 의미
0 은 빈칸, 1은 벽
### 로봇의 출발 위치
(1, 1) -> 가장 왼쪽 상단의 위치를 (1, 1) 이라고 본다. 
### 로봇
- 로봇은 2 X 1 크기이다
- 로봇은 회전가능하다. 두 칸 중 어느 방향으로도 회전이 가능하다. 
- 단, 회전 반경에 벽이 존재하면 회전할 수 없다.
