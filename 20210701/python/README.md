## 조건
### 릴레이션 이란?
- 관계형 데이터베이스의 테이블
### 후보키란?
- 튜플을 유일하게 식별하는 값
### 후보키의 조건은?
- 유일성 - 유일하게 식별가능해야함
- 최소성 - 후보키는 복수로 구성될수 있으며, 그 개수를 유일성을 만족하기 위한 최소개수로 하여야 한다는 뜻.
