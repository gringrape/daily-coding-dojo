# 1. 크레인 인형 뽑기

https://programmers.co.kr/learn/courses/30/lessons/64061

## 주어진 것

인형, 게임판, 바구니

### board

N x N 의 정사각 격자.

각 격자에는 인형이 들어있다. 

### moves

크레인을 작동 시킨 위치. 격자판의 가로.

moves( 세로 )( 가로 )

## 구하는 것

크레인을 모두 작동시킨 후 터트려져 사라진 인형의 개수

### 계획

- 인형을 든다. -> edge 인형이 위치에 없다
- 인형을 바구니에 넣는다. 
- 바구니에서 나란한 인형이 같으면 터진다.
- 인형이 터지면 카운트가 올라간다.
